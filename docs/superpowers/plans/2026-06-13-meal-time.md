# Meal Time Choice Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Remove the dietary filters (Vegetarian/Vegan/Gluten-free/Dairy-free) and replace them with a single-select Breakfast/Lunch/Dinner choice with a clock-based smart default (AUTO badge + explanatory hint), steering the AI's meal suggestions.

**Architecture:** `lib/filters.js` swaps `DIETARY_FILTERS` for `MEAL_TIMES` (id + emoji) plus a `defaultMealTime()` helper. `app/page.js` replaces the `filters` array state with `mealTime` (string|null) + `mealTimeAuto` (bool); the smart default is applied in a mount `useEffect` (NOT in `useState` — the page is statically prerendered, and a build-time clock value would cause a hydration mismatch with the visitor's local time). The shared hint string is computed once in page.js and passed to the mobile `DecideTab`. `/api/suggest` gains a validated `mealTime` prompt line; both API routes drop the dietary prompt block. The `recipe_history.dietary_filters` DB column stays and keeps receiving `[]`.

**Tech Stack:** Next.js 15 App Router, CSS Modules. **No git repo** — skip commits; verification gate is `npm run build` from `C:\Users\User\.local\bin\meal-decider`.

**Spec:** `docs/superpowers/specs/2026-06-13-meal-time-design.md`

- [ ] **Step 1: Replace the whole file**

```js
// Filter chip constants shared by the desktop page and mobile DecideTab.
export const MEAL_TIMES = [
  { id: 'Breakfast', emoji: '🍳' },
  { id: 'Lunch', emoji: '🥗' },
  { id: 'Dinner', emoji: '🍲' },
];
export const CUISINE_FILTERS = ['Asian', 'Italian', 'Turkish', 'Mexican', 'Mediterranean'];

// Smart default: which meal is the clock pointing at?
// 04:00–10:59 → Breakfast · 11:00–15:59 → Lunch · 16:00–03:59 → Dinner
export function defaultMealTime(date = new Date()) {
  const h = date.getHours();
  if (h >= 4 && h < 11) return 'Breakfast';
  if (h >= 11 && h < 16) return 'Lunch';
  return 'Dinner';
}
```
